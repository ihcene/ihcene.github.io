Cureety.com

Une plateforme de télésurveillance ville-hôpital pour transformer le parcours de soins des patients atteints d'un cancer.

Visitez le site

Une solution modulaire et interopérable.

La solution Cureety est une application moderne, dotée d’une interface simple d’utilisation, et conçue pour s’intégrer dans le quotidien de tous les établissements de soins, quelle que soit leur taille. Cureety est une application modulaire et peut ainsi être utilisée pour suivre les toxicités induites par tous types de traitements anti-tumoraux.
